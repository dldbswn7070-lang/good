##📝 Commit Message Rule
* **Python (첫 풀이)**: `[Python] 문제이름 / 난이도 / 소요시간`
* **Python (다른 풀이 추가)**: `[Refactor] 문제이름 / 새로운 방식 요약`
* **SQL**: `[SQL] 문제이름 / 유형`

Example: [Python] 짝수와 홀수 / Level4  / 3시간 or Review Needed(복습 필요)
